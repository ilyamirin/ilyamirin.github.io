`aquarium_nullclaw` — это control surface для дисциплинированного запуска agent systems.

Он объединяет runtime boundaries, secrets separation, gateway logic и human-operable UI. Здесь важен не только execution. Важна безопасная и понятная execution.

Что это доказывает обо мне:

- я умею мыслить control planes, а не только скриптами
- я понимаю operator needs, а не только model behavior
- я строю AI infrastructure, которую предполагается использовать и контролировать
