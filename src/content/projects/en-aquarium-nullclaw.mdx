`aquarium_nullclaw` is a control surface for running agent systems in a disciplined way.

It combines runtime boundaries, secrets separation, gateway logic, and a human-operable UI. The point is not only execution. The point is safe, legible execution.

What it proves about me:

- I can think in terms of control planes, not just scripts
- I understand operator needs, not only model behavior
- I build AI infrastructure that is meant to be used and supervised
