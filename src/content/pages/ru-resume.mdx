Я broad AI builder с необычным продуктовым вкусом и сильным системным уклоном.

Моя работа охватывает coding agents, evaluation harnesses, OpenAI-compatible infrastructure, runtime control planes, autonomous coding workflows и generative media pipelines. Общая линия здесь одна: operational clarity. Мне важно превращать модельные возможности в системы, которые выдерживают реальное использование, реальные ограничения и реальную проверку.

## В чём я особенно силён

- Проектирую AI-системы, которые связывают поведение моделей с полезными операторскими workflow
- Строю инфраструктуру и control surfaces вокруг LLM, а не отношусь к ним как к отдельным API
- Быстро перевожу прототипную энергию в production-minded структуру
- Умею не разрывать generative и creative work с systems quality, а собирать их в одну идентичность

## Публичные доказательства

- `agent-bench-lab`: benchmark harness для end-to-end сравнения coding agents
- `aquarium_nullclaw`: control plane для isolated AI runtimes
- `replicate_proxy`: OpenAI-compatible orchestration layer для multi-provider model access
- `autocoder` и `LoomPilot`: системы autonomous coding delivery
- `cashflow-underwriting-demo`: applied ML и business-facing risk product demo

## Текущий фокус

Сильные international roles, где одновременно важны AI systems, product judgment и implementation depth.
