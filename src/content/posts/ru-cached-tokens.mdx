Cached tokens — это точка, где работа с LLM перестаёт выглядеть как магия и начинает выглядеть как engineering.

Полезный урок здесь не только про billing. Он про архитектуру. Если статический prefix твоего prompt остаётся стабильным и детерминированным, система может переиспользовать вычисление вместо того, чтобы каждый раз пересобирать его заново.

Это заставляет смотреть на структуру prompt как на реальную инженерную поверхность:

- стабильный reusable context
- deterministic serialization
- чистое разделение между static и dynamic input

Сильные LLM integrations определяются не только качеством модели, но и дисциплиной reuse computation.
